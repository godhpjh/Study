## 문법

### @Entity
- 데이타베이스의 테이블과 일대일로 매칭되는 객체 단위
- Entity 객체의 인스턴스 하나가 테이블에서 하나의 레코드 값
- 먼저 Spring Boot 를 설정할때 spring.jpa.hibernate.ddl-auto 설정이 create 혹은 update 로 되어 있을 경우 Spring 프로젝트가 시작될때 EntityManager 가 자동으로 DDL 을 수행해 테이블을 생성
- name 속성을 이용해 데이타베이스상의 실제 테이블 명칭을 지정하지 않는다면 Entity 클래스의 이름 그대로 CamelCase 를 유지한채 테이블이 생성
- UnderScore 를 이용한 테이블 명명 규칙을 이용하는 것이 좋습니다

```java
@Entity
public class Organization {
    ...
}

@Entity(name = "employee")
public class Employee {
    ...
}

@Entity(name = "employee_address")
public class EmployeeAddress {
    ...
}
```
